Definition : 

- Les Signaux : 
    
    . Mecanisme de communication a base de messages
    . Message = nombre entre 1 et 31
    . Ordre de reception aleatoire 
    . Perte possible en cas d'envoie multiple du meme numero de sinal

    exemple : - "kill -sig pid" : envoie un signal "sig" a "pid"
              - "trap 'expr' sig" : associe "expr" q la reception d'un signal "sig"

- Processus : 

    . Un programme existe sur un disque dur un Processus en memoire vive
    








Commandes :

    . ps -e : permet de voir tous les processus actifs sur le systeme unix
